## Introduction ([‰∏≠Êñá](/README_zh.md))

app_injector is a dynamic library injector for application. It support the following applications:

* Android APK

It will support in the future:

* iPhoneOS IPA
* Windows Program
* Linux Program
* MacOS Program

If you want to know more, please refer to:

* [Documents](https://xmake.io/#/home)
* [HomePage](https://xmake.io)
* [Github](https://github.com/lanoox/app_injector)
* [Gitee](https://gitee.com/lanoox/app_injector)

## Prerequisites

XMake installed on the system. Available [here](https://github.com/xmake-io/xmake).

## Build

```console
$ xmake
```

## Installation

```console
$ xmake install
```
